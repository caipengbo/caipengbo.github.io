2019年3月份随笔

Mar 1, 2019 | 随笔

3/1/2019 周五

今天是3月的第一天，昨晚本来打算早点休息，结果又没有控制住自己，看电视看到深夜！！！一定要克制自己（增强自己的自控能力）！！！

早晨继续学习MIT线性代数，由于昨晚睡得很晚，早晨起得很早，中午的精神状态极差！！

下午开会浪费了很多的时间，但是还是抽空把下周的机器学习内容大纲整理了一下，机器学习部分的任务如期完成了，下周内容比较多，加上需要加强CoOper系统的工作，可能要拖延该部分的进度。不过，还是尽量提高效率吧！！晚上刷题，效率也是奇差无比，自己理解错了算法的细节，导致出了很难发现的Bug浪费了不少的时间，现在感觉自己写一个高效的算法好慢好慢，慢慢提高吧！！加油！！！

明天是周末了，学习的线性代数和机器学习的知识要复习、整理一下（整理到Anki）！！

3/2/2019 周六

上午学习了线性代数、整理了Anki，效率还可以，下午效率及其低下，做了很多无关的事，浪费了很多时间，晚上调了很久的bug，还是没有解决！打算明天解决问题，处理一下原始数据。

3/3 周日

早晨学完了MIT第七课，参加了一场 LeetCode Weekly Contest,感觉还行，只不过还是很久没编程的原因，手生，对于一些算法的实现能力下降了很多，以后多练练应该会恢复的，以后每周日都要参加！！！

下午又去游泳啦，进步也很大，可以憋气、一次换气了，连续换气还是不太熟练，下周把连续换气，手脚协调练好！！

晚上整理了早晨学习的线性代数的笔记，消化了一下知识点，调了很久的Bug还是没有找到原因，唉，无奈啊！！

今天早晨的效率还是可以的，下午还是纠结了很多无关的东西，浪费了很多的时间，明天是新的一天，还是希望不断提高自己的效率！！！加油！！！

3/5 周二

早晨学习了MIT线性代数第9课，第9课的内容比较简单，所以就复习了一下第7、8课的内容。中午12点到下午2点，睡了半小时，依然浪费了一个半小时的时间（这段时间一定要充分利用起来，今后一定要改正！）。

下午又重新思考了一下自己要做的研究 —— 基于时间序列的Stack Overflow社区热点演化和预测（利用的大数据分布式运算等技术，尽可能快速的给出结果）。

晚上搭建了ElasticSearch搜索引擎, 学习了一下ES的基本知识，并知道了如何使用ES的 Python接口。接下来两天要好好处理一下数据，早日将数据上传到ES之中。

今天对大数据、分布式思考了一下，感觉分布式并不是那么的神秘，尝试去搞一下分布式（Spark）！！！今天要刷的剑指offer题又没有刷，已经连续两天没有刷题了，明天一定要刷一下！！！

3/11 周一

今天作为OS助教，去了一趟沙河监考学生上机，从早晨10点半一直待到傍晚6点，浪费了很多的时间，下次提前去要拿些书去看！

晚上把导入数据到ES的代码写完了，写了很久（找了半天通俗易懂的代码，为此浪费了很多的时间），以后一定不能只会吃现成的东西，要从源头解决问题，搞懂出了什么问题，要敢于直面问题，不能一味的去Google或者Baidu现成的东西！！！

今天还把正则化的东西看了看，还是需要再消化一下，只是懂得了道理，公式以及个人陈述还是不会！

3/12 周二

今天早晨上了一课MIT线性代数，然后搞了一天的机器学习逻辑回归和Softmax回归，效率依然很低。

又重新审视了一下自己的学习方法，我觉得先不要把面试的那些细节准备的那么完全（因为就算现在记住了，到时候也会忘记的），当前先把不懂的知识点全部搞懂，需要记忆的部分（公式推导，细节比较，面试细节）以后再去背诵记忆。

当前先把：集成学习、BP神经网络模型（为深度模型做准备）、关联关系学习这些东西搞懂，LDA主题模型（代码推荐项目）、学习Hadoop、Spark集群（5台机器），为自己的项目做准备。

对于MIT线性代数，也是选择自己需要的内容，有详有略的去听，去记。

切忌浮躁贪快

3/13 周三

早晨有选择的学了MIT线性代数课程，明天将会学习很重要的一课（一定要认真听讲）。

下午学习了集成学习的相关知识，分清了学习的主次，明天上午上一会儿课，下午的自然辩证法打算翘课去游泳！！！

晚上游泳回来打算修改代码RecommendService代码，然后刷两道题！！！（一定要刷,不刷不回宿舍）

3/16 周六（颐和园半日游）

前天周四，下午游了泳，晚上刷了一道题，效率依旧很低。

昨天周五，下午去打了球，晚上效率依然低！！！自己学习很容易走神，不能注意力集中的思考复杂的知识（公式、复杂的概念思想），知识难，消化的慢，可以理解，但是对于难的东西思考的时候很容易陷入走神的状态（这一点一定要改）！！！

今天周六，下午和实验室三位同学去了颐和园游玩，然后在中关村购物广场吃了大盘鸡，晚上回来实验室已经7点有余了，晚上花了一小时多写了一道关于二叉树的题，依旧不熟练，不能专心！！（改、改、改）

明天早晨10点要参加LeetCode Weekly Contest ；早晨到实验室到10点这段时间会整理MIT线性代数的知识，然后有详有略的学完正交矩阵（应该能完成）！

下午专心搞java编程助手的插件。

3/18 周一

昨天LeetCode Weekly Contest虽然只做了两道题，但是明显感觉自己进步了，要坚持刷剑指offer，刷完所有的60多道题，然后开始进军LeetCode Medium。

今天早晨把行列式相关的知识整理完毕了，感觉自己效率提高了不少，有些进步，加油哦！！！下午把Java编程助手插件调试了80%，希望明天一鼓作气，把这个彻底搞完！！！

写完这边随笔继续刷剑指offer 喽！！

3/21 周四

今天早晨整理了特征值与特征向量的内容，就去上课了。

下午调试Java插件代码，明天继续弄，希望能解决问题。然后看了很多GBDT的资料，明天一早整理完毕。

晚上因为GBDT的知识，没能刷题，明天中午11点左右刷一刷，晚上刷一刷！！弥补这两天的缺失！！！

3/22 周五

今天终于把插件成功运行起来了！！！炒鸡开心！！！

早晨把GBDT搞懂了，MIT特征值部分也整理完毕，明天整理正定性（线性代数部分快结束了）！！

中午做了一道关于二分查找的题，感觉自己又进步了一些！！！

下午开会俩小时（浪费时间），回来把插件修改了，也做了一些细节的修改，花少部分的时间，将ES迁移到阿里云、把服务挂到阿里云、把插件下载挂到阿里云即可！！！

晚上搞一搞XGBoost的知识！！！
